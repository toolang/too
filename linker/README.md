# 链接测试demo
因为compiler和asmer还未自举完成，目前需要靠gcc生成elf对象文件
```
$ cd linker/demo
$ gcc -c *.s -static -nostdlib -e main
```
链接生成可执行文件生成可执行文件
```
$ cd linker
$ ./bin/amd64_linux -p ./demo
$ chmod 777 a.out

$ ./a.out
> hello toolang!
```
